# Прогнозирование снижения активности клиентов (Churn Prediction)

## Постановка задачи
Интернет-магазин «В один клик» столкнулся со снижением покупательской активности постоянных клиентов. 
**Цель проекта:** разработать модель машинного обучения для прогнозирования вероятности снижения активности клиента в ближайшие 3 месяца и выявить ценные сегменты для целевого маркетингового удержания.

## Используемые модели
Для решения задачи бинарной классификации были обучены и сравнены четыре алгоритма:
*   KNeighborsClassifier
*   DecisionTreeClassifier
*   LogisticRegression
*   **SVC (Support Vector Classifier)** — выбран как лучший.

Для каждой модели был построен единый пайплайн предобработки (кодирование категориальных признаков, масштабирование числовых) и выполнен поиск гиперпараметров (RandomizedSearchCV).

## Результат применения
*   **Лучшая модель:** SVC показала наивысшую точность на тестовой выборке (**ROC-AUC = 0.9141**).
*   **Сегментация:** Выделен критический сегмент из **353 клиентов** с высоким риском оттока (≥ 0.6) и высокой прибыльностью (≥ 3 у.е.).
*   **Бизнес-эффект:** На основе прогнозов сформулированы рекомендации по удержанию: повышение стабильности сервиса, оптимизация оформления заказа, персонализированные акции и проактивная коммуникация. Модель позволяет не просто предсказать отток, но и предотвратить его, сохраняя лояльность прибыльных клиентов.
